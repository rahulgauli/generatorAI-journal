# GeneratorAI - 
Uses large language models to develop video clips using daily news.

#### Pre-req:
- Python 3.11
- Pipenv
- .env file 

#### How to Run: 
- get the CNN newsAPI and store it in your .env file.
- run the following Command:

``` shell
pipenv install
pipenv install --dev
pipenv run python3 -m app.main
```
